Build and Run Instructions

make
cd bin
./sim02 config_<#>.conf

Where '<#>' in ./sim02 config_<#>.conf is the configuration number of the desired file.

Additionally, there must be an <A> folder in bin if a config file calls for the output file to be in <A>/someoutputfile.out for example.

Dependancies

Required dependancies for Sim2:

Ubuntu

sudo apt-get install libboost-all-dev

### macOS ###

brew install boost

It has been my experience that the CFLAG variable parameter in the Makefile must match the library location in your computer's directory. However, while testing on Ubuntu.cse.unr.edu I was proved wrong for what that's worth.
